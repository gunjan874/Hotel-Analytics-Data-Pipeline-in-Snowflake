# Hotel-Analytics-Data-Pipeline-in-Snowflake
Hotel Analytics Dashboard project that cleans and standardizes raw booking data to deliver clear business insights. It provides monthly revenue and booking trends, city-wise performance, booking type and status analysis, and key KPIs through interactive visualizations to support fast, data-driven decision making.
